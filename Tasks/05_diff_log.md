# Задача 05 — `git diff` и `git log`

## Новые команды

```bash
git log --oneline --graph --all   # граф истории всех веток
git diff                           # изменения не добавленные в staging
git diff --staged                  # изменения добавленные в staging
git diff main feature/damping      # сравнить две ветки
git show <хэш>                     # посмотреть конкретный коммит
```

`log` и `diff` — это инструменты навигации по истории.
Они не изменяют ничего, только показывают. Пользоваться ими нужно постоянно.

---

## Физическая модель

В вашей модели пока нет затухания — система колеблется вечно.
В реальности любая струна теряет энергию: через воздух, через крепления.
В этой задаче добавим компоненту которая отвечает за затухания и научимся видеть разницу между версиями через `diff`.

Уравнение с затуханием:

$$\ddot{x} = F_{spring} - \gamma \dot{x}$$

где γ — коэффициент затухания.

---

## Задание

### Шаг 1 — Создать ветку для эксперимента

```bash
git checkout -b feature/damping
```

### Шаг 2 — Добавить затухание

В файле `physics.py` в функции `process_particles()` добавьте параметр `damping`:

```python
def process_particles(parts, springs, dt, fixed, damping=0.0):
    ...
    accs = <существующая формула>
    vlcs = vlcs + accs * dt - damping * vlcs * dt  # член затухания
    crds = crds + vlcs * dt
```

Пробросите параметр через `time_process()` и `string()`.

### Шаг 3 — Изучить изменения через diff

До того как делать коммит — посмотрите что именно изменилось:

```bash
git diff
```

После `git add -p`:

```bash
git diff --staged
```

### Шаг 4 — Сравнить ветки

```bash
git diff main feature/damping
```

Вы увидите все отличия ветки с затуханием от основной.

### Шаг 5 — Изучить историю графом

```bash
git log --oneline --graph --all
```

Вы увидите, как ветки расходятся от общего предка.

### Шаг 6 — Физический эксперимент

Запустите симуляцию при трёх значениях:
- `damping=0.0` — без затухания
- `damping=0.1` — слабое затухание  
- `damping=5.0` — сильное затухание

Опишите наблюдения в комментарии к коммиту.
