# Задача 09 — `git stash`

## Новая команда

```bash
git stash                  # спрятать текущие изменения
git stash list             # список спрятанных изменений
git stash pop              # достать последнее спрятанное
git stash apply stash@{1} # достать конкретное
git stash drop             # удалить запись из стека
```

`stash` — временное хранилище для незакоммиченных изменений.
Аналог стикера «вернусь к этому позже»: вы откладываете незаконченную работу,
переключаетесь на срочную задачу, затем возвращаетесь и продолжаете.

---

## Физическая модель

Вы исследуете фигуры Хладни на мембране из предыдущей задачи.
В середине работы оказывается, что в `main` есть баг — симуляция зависает при `num > 50`.
Нужно срочно переключиться и починить, но текущая работа ещё не готова к коммиту.

## Задание

### Шаг 1 — Начать работу над фигурами Хладни

Создайте ветку и начните эксперимент:

```bash
git checkout -b feature/chladni
```

Добавьте в код функцию возбуждения мембраны на заданной частоте:

```python
def excite_membrane(parts, frequency, amplitude, t):
    """Возбуждает мембрану синусоидальной силой"""
    crds, vlcs, accs, mass, num = parts
    # точка возбуждения — центр
    center = num // 2
    vlcs[center][1] += amplitude * np.sin(2 * np.pi * frequency * t)
    return (crds, vlcs, accs, mass, num)
```

Код незаконченный — функция есть, но ещё не подключена к основному циклу.

### Шаг 2 — Спрятать незаконченную работу

```bash
git stash -m "незаконченное: возбуждение мембраны на частоте"
git status  # рабочая директория чистая
```

### Шаг 3 — Срочно переключиться и починить баг

```bash
git checkout main
# найти и починить проблему с большим числом частиц
git add .
git commit -m "fix: устранено зависание при num > 50"
git push origin main
```

### Шаг 4 — Вернуться к фигурам Хладни

```bash
git checkout feature/chladni
git stash pop
git stash list  # убедиться что стек пуст
```

### Шаг 5 — Завершить эксперимент

Подключите `excite_membrane()` к циклу. Запустите на нескольких частотах.
Найдите точки где амплитуда близка к нулю — это и есть узловые линии фигур Хладни.
Закрасьте эти точки на `imshow`-графике.

## Подсказки

- `git stash` не трогает неотслеживаемые файлы. Чтобы спрятать и их: `git stash -u`
- Стек stash — это именно стек: `pop` достаёт последнее. Используйте `stash@{N}` для конкретного
- `stash` — не замена веткам. Для долгосрочного хранения лучше создать ветку
