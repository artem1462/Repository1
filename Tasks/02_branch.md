# Задача 02 — `git branch` и `git checkout`

## Новые команды

```bash
git branch <название>       # создать ветку
git checkout <название>     # переключиться на ветку
git checkout -b <название>  # создать и сразу переключиться
git branch                  # посмотреть список веток
```

Ветка — это независимая линия разработки. Всё, что вы делаете в ветке,
не затрагивает основной код в `main` до тех пор, пока вы сами этого не захотите.

---

## Физическая модель

В вашем коде начальная скорость второй частицы задана жёстко:

```python
parts[1][1][1] = 300
```

Это неудобно: чтобы попробовать другие начальные условия, приходится менять код вручную.
Давайте попробуем параметризовать начальные условия — не ломая рабочую версию.
Именно для этого и нужны ветки: экспериментируете в ветке, основной код остаётся нетронутым.

---

## Задание

1. Убедитесь, что вы находитесь в ветке `main`:

```bash
git branch
```

2. Создайте новую ветку для эксперимента:

```bash
git checkout -b feature/initial-conditions
```

3. В функции `string()` замените хардкод начальной скорости на параметр:

```python
def string(num, length, tense, k, t, math_dt, anim_dt,
           anim_speed=1, anim_show={}, fixate_edges=True,
           initial_velocity=300, excited_particle=1):
    ...
    parts[1][excited_particle][1] = initial_velocity
```

4. Убедитесь, что симуляция по-прежнему запускается

5. Переключитесь обратно на `main` и убедитесь, что там ничего не изменилось:

```bash
git checkout main
```

---

## Подсказки

- Название ветки `feature/...` — это соглашение, не требование Git. Такой стиль принят в большинстве проектов
- `git branch` со звёздочкой показывает текущую ветку
- Переключаться между ветками можно сколько угодно раз — изменения сохраняются в каждой независимо

---

## Ожидаемый результат

- Существуют две ветки: `main` и `feature/initial-conditions`
- В ветке `feature/initial-conditions` функция принимает параметры начальных условий
- В ветке `main` код остался прежним
- Ну и нужно постараться дать обяснение и понимать зачем нужны ветки в научном проекте?
